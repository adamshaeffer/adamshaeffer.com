Roll-A-Ball

Controls:

WASD / Arrow keys - Move
Collect all 12 cubes to win!

About the game:

Play this super simple, one level 3D Unity game! Collect all 12 cubes to win! I made this game by following a Unity tutorial making a very simple rolling ball game. I added the challenging and moving elements to make it my own and make it a little more enjoyable. I apologize if you wanted more, the file to edit was lost by accident.
